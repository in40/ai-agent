LangGraph Visual Editor Suite

This suite provides multiple interfaces for visualizing and editing your LangGraph workflows:

  • Streamlit Editor: Browser-based editor with visualization and simulation
  • React Editor: Modern UI with drag-and-drop capabilities
  • LangGraph Studio: Professional development environment (requires separate installation)

1. Enhanced Streamlit Editor

A comprehensive visual editor built with Streamlit featuring workflow visualization, editing, simulation, and debugging capabilities.

Open Streamlit Editor

2. React Flow Editor

A modern, interactive editor built with React Flow for creating and editing LangGraph workflows with drag-and-drop functionality.

Open React Editor

3. LangGraph Studio Integration

Professional development environment for LangGraph with advanced debugging and visualization features.

Note: Requires separate installation of LangGraph Studio (currently only available for Apple Silicon Macs).
